# Hello World KCL Module Test

This is a simple "Hello World" KCL module created for testing purposes.

## Files

- `kcl.mod` - Module configuration file
- `main.k` - Main KCL file containing the hello world example

## Running the Module

```bash
kcl main.k
```

This will output:
```
Hello, World!
```

## Structure

The module demonstrates:
- Basic KCL package definition
- Simple string variable assignment
- Configuration object creation
- Print statement usage
